2.1장 인공 뉴런: 초기 머신 러닝의 간단한 역사
퍼셉트론 이전 초창기 머신러닝
ㄴ>맥컬록-피츠 뉴런(맥컬록과 윌터피츠가 고안한 간소화된 뇌의 뉴런개념발표)
      ㄴ>신경세포를 이진 출력을 내는 간단한 논리 회로 표현

프랑크 로젠블라트가 고안(MCP뉴런모델 기반) -> 퍼셉트론 학습개념
퍼셉트론 규칙 -> 자동으로 최적 가중치(입력특성에 곱하는 계수) 학습하는 알고리즘 제안
 ㄴ>지도학습, 분류개념 -> 한 클래스 속하는지 아닌지 예측

인공뉴런 아이디어
ㄴ> 입력 값 x와 가중치 벡터 w의 선형 조합 결정함수(단위 계단 함수 변형) 정의	
      ㄴ> 임계값보다 크면 1 작으면 0
머신러닝 분야서 음수 임계 값 또는 가중치를 절편으로 정의

퍼셉트론 학습규칙 -> 환원주의 접근방식 사용
1. 가중치를 0 또는 랜덤한 작은값으로 초기화
2. 각 훈련 샘플 x에서 다음 작업 -> 출력 값 y계산, 가중치 업데이트
-> 출력값 = 단위계산함수로 예측한 클래스 레이블
-> 개별가중치 동시 업데이트
가중치에 대한 업데이트 값 = 학습률 * (진짜 클래스 레이블 - 예측 클래스 레이블) * 훈련 샘플

퍼셉트론 클래스 레이블을
정확히 예측한 경우 => 가중치 변경 X
잘못 예측한 경우 => 가중치 양성 또는 음성 타깃 클래스 방향

두 클래스를 선형 결정 경계로 나눌 수 없다면 훈련 데이터셋을 반복할 최대 횟수 지정
