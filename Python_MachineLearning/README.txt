Library
How to Download(cmd) and Why we need it

Pandas
essential for statistics, data science and machine learning.
Downlaod : pip install pandas

NumPy
short term of Numerical Python which is the representative of python numerical analysis library.
especially it provides multi-dimensional array and array calculation when it comes to linear
algebra calculation.
Download : pip install numpy

Matplotlib
Matplotlib is a 2D visualization tool.
Download : pip install matplotlib

Scipy
Scipy has variety of package used for scientific calculation. (differential and integral calculus
, linear algebra, matrix calculation, equation)
Download : pip install scipy

scikit-learn
scikit-learn provides machinelearning models used at practical affairs such as regression analysis,
classification, crowding.
Download : pip install -U scikit-learn
