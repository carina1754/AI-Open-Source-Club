1.2장 머신 러닝의 세가지 종류
*지도학습: 데이터 레이블O, 직접 피드백 O, 미래 예측
분류와 회귀로 구분됨
ㄴ>분류(클래스레이블을 예측하는 것)
I   ㄴ> 이진 분류(True/False로 구분): 음성/클래스를 나누는데 결정경계라는 것으로 구분
I   ㄴ> 다중 분류(두개이상 클래스 레이블 구분): 샘플들을 모아서 훈련 데이터셋 구성으로 구분(예:손으로 쓴 글자 인식)
ㄴ> 회귀분석: 예측 변수와 연속적 반응 변수가 주어졌을 때 연속적 출력값 예측
     ㄴ> 선형회귀: 특성x와 타깃 y가 주어지면 데이터 포인트와 직선 사이 거리 최소가 되는 직선 긋고 그 기울기와 절편
                       이용으로 새로운 데이터 출력값 예측
*강화학습: 데이터 레이블X, 피드백 X, 숨겨진 구조 찾기
환경과 상호작용 - 시스템(에이전트) 성능 향상
보상신호 포함: 강화학습 - 지도학습
피드백: 정답 레이블 또는 값 X, 얼마나 행동이 좋은 측정한 값O
강화학습 예시: 체스(체스판의 상태에 따라 기물의 이동 결정)
*비지도학습: 결정 과정, 보상시스템, 연속된 행동서 학습
알려진 출력 값이나 보상 함수의 도움을 받지 않고 데이터 구조 탐색 O
- 군집, 차원 축소가 있음
ㄴ> 군집: 사전 정보 없이 쌓여 있는 그룹 정보를 의미 있는 서브 그룹 또는 클러스터 조직/탐색적 데이터 분석 기법
ㄴ> 차원축소: 고차원 데이터를 알고리즘 계산성능과 저장공간의 한계 극복을 위해 데이터를 압축

1.3장 기본 용어와 표기법 소개
훈련샘플: 데이터셋을 나타내는 테이블의 행(row)
훈련: 파라미터 추정
특성: 데이터 테이블이나 행렬의 열(column) / x축
타깃: 결과, 레이블 / y축
손실함수: 이를 하나의 데이터 포인터에 대해 측정한 손실로 사용

1.4장 머신 러닝 시스템 구축 로드맵
전처리 -> 학습 -> 평가 -> 예측
머신러닝 알고리즘이 새 데이터에서 일반화 되는 지 판단: 훈련 데이터셋(머신러닝 훈련용)/테스트 데이터셋(최종 모델 평가용)
정확도: 정확히 분류된 샘플비율
하이퍼파라미터: 데이터 학습하는 파라미터가 아니라 모델 성능을 향상위해 사용하는 다이얼

1.5장 머신 러닝을 위한 파이썬
파이썬 -> 과학 컴퓨팅, 머신 러닝을 위한 유용한 라이브러리 많음
       ㄴ> 넘파이와 사이파이 같은 라이브러리 -> 다차원 배열에서 벡터화된 연산을 빠르게 수행
NumPy,SciPy,Matplotlib,Pandas 등 패키지 사용
